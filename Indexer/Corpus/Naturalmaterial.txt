<p>A <b>natural material</b> is any product or physical matter that comes from plants, animals, or the ground. <a class="mw-redirect" href="/wiki/Minerals" title="Minerals">Minerals</a> and the <a class="mw-redirect" href="/wiki/Metals" title="Metals">metals</a> that can be extracted from them (without further modification) are also considered to belong into this category. Natural materials are also often used in textiles. Types include:</p>
<ul>
<li><a href="/wiki/Biotic_material" title="Biotic material">Biotic materials</a>
<ul>
<li><a href="/wiki/Wood" title="Wood">Wood</a> (<a href="/wiki/Rattan" title="Rattan">rattan</a>, <a href="/wiki/Bamboo" title="Bamboo">bamboo</a>, <a href="/wiki/Bark" title="Bark">bark</a> etc.)</li>
<li><a href="/wiki/Natural_fiber" title="Natural fiber">Natural fiber</a>(<a href="/wiki/Silk" title="Silk">silk</a>),(<a href="/wiki/Wool" title="Wool">wool</a>, <a href="/wiki/Cotton" title="Cotton">cotton</a>, <a href="/wiki/Flax" title="Flax">flax</a>, <a href="/wiki/Hemp" title="Hemp">hemp</a>, <a href="/wiki/Jute" title="Jute">jute</a>, <a href="/wiki/Ceiba_pentandra" title="Ceiba pentandra">kapok</a>, <a href="/wiki/Kenaf" title="Kenaf">kenaf</a>, <a href="/wiki/Moss" title="Moss">moss</a>, etc.)</li>
</ul>
</li>
<li>Inorganic material
<ul>
<li><a href="/wiki/Rock_(geology)" title="Rock (geology)">Stone</a> (<a href="/wiki/Flint" title="Flint">flint</a>, <a href="/wiki/Granite" title="Granite">granite</a>, <a href="/wiki/Obsidian" title="Obsidian">obsidian</a>, <a href="/wiki/Sandstone" title="Sandstone">sandstone</a>, <a href="/wiki/Sand" title="Sand">sand</a>, <a href="/wiki/Gemstone" title="Gemstone">gems</a>, <a href="/wiki/Glass" title="Glass">glass</a>, etc.)</li>
<li><a href="/wiki/Native_metal" title="Native metal">Native metal</a> (<a href="/wiki/Copper" title="Copper">copper</a>, <a href="/wiki/Bronze" title="Bronze">bronze</a>, <a href="/wiki/Iron" title="Iron">iron</a>, <a href="/wiki/Gold" title="Gold">gold</a>, <a href="/wiki/Silver" title="Silver">silver</a>, etc.)</li>
<li><a href="/wiki/Composite_material" title="Composite material">Composites</a> (<a href="/wiki/Clay" title="Clay">clay</a>, <a href="/wiki/Porcelain" title="Porcelain">porcelain</a>, <a href="/wiki/Plasticine" title="Plasticine">plasticine</a>, etc.)</li>
</ul>
</li>
<li>Other natural materials.
<ul>
<li><a href="/wiki/Soil" title="Soil">Soil</a></li>
</ul>
</li>
</ul>
